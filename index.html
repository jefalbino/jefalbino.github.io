JSmart Web Framework

Web application framework integrated with Bootstrap v3 for developing responsive, mobile first projects on the web with smooth server integration

Getting Started

An open source Java and JavaScript based web application framework

JSmart is a powerful, open source, web framework for the Java platform aimed to improve developers productivity and experience thanks to easy and centralized way to setup the application, variety of web components integrated with Bootstrap v3.3.5, action based programming, asynchronous programming and integration with Spring 4 and EJB 3.

Powerful components

Variety of built-in components support, based on custom tag libs to generate HTML elements with CSS3. All you need to build modern web applications.

Bootstrap integration

Action based components integrated with Bootstrap v3.3.5 to provide great look and feel following the standards of HTML5 and CSS3 for mobile first web applications.

Smooth Java integration

Seamlessly and transparently integrates with Java Web and Application containers, such as Tomcat, Jetty, JBoss and Glassfish.

Spring and EJB support

Integrated support for EJB 3 and Spring 4 frameworks for dependency injection and much more. All you need to speed up your development and provide latest technology features.
